# 🚀 QUICK START - Gemini AI cho FasTrack ERP

## Cài đặt trong 3 bước

### 1. Cài đặt thư viện (1 phút)
```bash
pip install google-generativeai
```

Hoặc chạy (Windows):
```bash
SETUP_GEMINI_AI.bat
```

### 2. Lấy API Key (2 phút)
1. Mở: https://aistudio.google.com/app/apikey
2. Đăng nhập Google (nếu chưa)
3. Nhấp **"Create API Key"**
4. Sao chép API Key (giữ an toàn)

### 3. Cấu hình trong ứng dụng (1 phút)
1. Chạy: `python main.py`
2. Menu bên trái → **"🤖 Trợ lý AI"**
3. Nhấp **"⚙ Cấu hình API"**
4. Dán API Key
5. Nhấp **"Kết nối & Lưu"**

✅ **Hoàn tất!** Bây giờ bạn có thể chat với AI!
